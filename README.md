## Run Locally<img 


**Prerequisites:**  Node.js

1. Rename folder to chronos-archives after extraction to prevent any issues
2. Install dependencies:
   `npm install`
3. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
4. Run the app:
   `npm run dev`
